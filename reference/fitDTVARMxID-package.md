# fitDTVARMxID: Fit the Discrete-Time Vector Autoregressive Model for Multiple Individuals

Fit the discrete-time vector autoregressive model for multiple
individuals using the 'OpenMx' package (Hunter, 2017
[doi:10.1080/10705511.2017.1369354](https://doi.org/10.1080/10705511.2017.1369354)
).
